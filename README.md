# blog0425

GitHub Pages blog using the [Just the Docs](https://github.com/just-the-docs/just-the-docs) Jekyll theme.

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

The local site runs at `http://localhost:4000/blog0425/`.

## GitHub Pages

In the repository settings, go to **Settings > Pages > Build and deployment** and set **Source** to **GitHub Actions**.
